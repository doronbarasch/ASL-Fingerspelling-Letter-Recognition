ASL Fingerspelling Letter Recognition

Collaborators: Doron Barasch, RanDair Porter

For this project we decided to take on the task of creating a model that would be able to interpret/translate standard ASL finger spelling letters in real time.
This proved to be rather difficult, requiring us to combine large datasets to create a model that would generalize well and accept, hopefully, as many correct finger letters as possible. As such, we combined all of the images, bar a test set from the second dataset, from three datasets.
These datasets can be found at the following links:
